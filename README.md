# Vending Machine using Verilog HDL

## Description
This project implements a vending machine using Finite State Machine (FSM) in Verilog HDL.

## Features
- Accepts coins
- Product selection logic
- Change return mechanism

## Tools Used
- Vivado Design Suite

## Files Included
- Vending_machine.v
- Vending_machine_tb.v
